=== Template Universidade ===
Template Universidade é um tema Wordpress criado por Lucas Sandes, a partir do starter-theme DevDmBootstrap3 de Danny Machal.

Este é um tema destinado a desenvolvedores, que pode ser utilizado como framework para o desenvolvimento de sítios de universidades públicas acessíveis e com boa usabilidade.

== Requisitos Wordpress ==
Requer pelo menos: 4.1
Testado até: 4.2.2


== Características ==
*  Layout responsivo (baseado em Bootstrap 3)
* Tema acessível, verificado pelo AccessMonitor (http://www.acessibilidade.gov.pt/accessmonitor/)


== Instalação ==
1. No painel administrativo, vá em Aparência -> Temas e clique no botão “Adicionar Novo”
2. Cliquem em “Fazer upload do tema” e selecione o arquivo .ZIP. Clique no botão “Instalar agora”
3. Clique em “Ativar” para usar o tema instalado.


== Ajustando a página inicial ==
1. No painel administrativo, vá em Páginas->Adicionar Nova
2. Dê um título para essa página nova (Página Inicial, por exemplo)
3. Na barra da direita, em Atributos de página, escolha o modelo Index
4. Clique em ”Publicar”

5. No painel administrativo, vá em Aparência->Personalizar
6. Clique no item “Páginia Inicial Estática” e selecione “Uma página estática”
7. Em página inicial, escolha a página que acabastes de criar (Página Inicial)
8. Clique em “Salvar & Publicar”

== Categorias necessárias para um bom funcionamento ==
Para desfrutar de todas as características do tema, é necessário criar três categorias.
1. No painel administrativo, vá em Posts->Categorias
2. Crie três categorias, com os seguintes nomes e slugs, respectivamente:

Nome: Notícias   /  Slug: noticias
Nome:banner   /  Slug: banner
Nome:Destaques   /  Slug:  destaques


== Configurando Banner Principal ==
1. No painel administrativo, vá em Posts->Adicionar Novo
2. Preencha o Título, conteúdo e o Resumo. Na página inicial aparecerá o título e o resumo, como linha fina
3. Do lado direito, no box “Imagem Destacada”, clique em “Configurar imagem destacada”
4. Selecione a imagem em Biblioteca de Mídia ou envie uma nova em “Enviar arquivos”.
5. Clique em “Configurar imagem destacada”

6. No box “Categorias”, escolha (pelo menos) a opção “banner”
7. Clique em “Publicar”


== Utilizando o conteúdo de exemplo ==
1. No painel administrativo, vá em Ferramentas->Importar
2. Clique em Wordpress e, em seguida, “Instalar agora”
3. Clique em Ativar Plugin & Importar
4. Clique em “Escolher arquivo”, e selecione o arquivo conteudo-exemplo.xml
5. Clique em “Fazer upload de arquivo e importar”
5a. Se você quiser, pode importar também as imagens selecionando “Download and import file attachments”
7. Clique em “Submit”
